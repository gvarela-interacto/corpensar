Sobre el procesamiento de PDF

El sistema utiliza inteligencia artificial (Google Gemini) para extraer información del PDF y crear automáticamente una caracterización municipal. El proceso:

  1. Extrae texto del PDF
  2. Procesa el texto con IA para identificar datos relevantes
  3. Crea o encuentra el municipio correspondiente
  4. Genera una caracterización con los datos encontrados
